--- layout: post title: Overriding default id in mongoid date: '2014-11-15T07:20:00.000-08:00' author: Chen Kinnrot tags: - mongodb - mongoid - Ruby modified_time: '2014-11-15T07:20:24.581-08:00' blogger_id: tag:blogger.com,1999:blog-8908829452516573489.post-4084665667764029032 blogger_orig_url: http://kinnrot.blogspot.com/2014/11/overriding-default-id-in-mongoid.html --- This is an easy one, just want to make sure it's clear.

Add the following field to your model:


field :_id, type: Moped::BSON::ObjectId

And to allow setting it:

1 def id=(id)
2   self[:_id] = id
3 end


Now you can just do obj.id=[Some ObjectId] and save the object.
