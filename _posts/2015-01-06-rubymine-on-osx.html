--- layout: post title: Rubymine on OSX memory issues date: '2015-01-06T13:32:00.001-08:00' author: Chen Kinnrot tags: - OSX - memory leak - Ruby - Rubymine modified_time: '2015-01-06T13:32:50.639-08:00' blogger_id: tag:blogger.com,1999:blog-8908829452516573489.post-3878550271401009642 blogger_orig_url: http://kinnrot.blogspot.com/2015/01/rubymine-on-osx.html --- I found this only today, you should run Rubymine as 32 bit process.
It works much faster and consume half the memory.

To set this got to the Applicaitons folder right click on rubymine->get info and check the "open in 32 bit mode"

Thats all!
